# Веб‑приложение для управления карточками


Простое и удобное веб‑приложение, позволяющее создавать, просматривать и управлять персональными карточками с информацией.

## Описание


Приложение позволяет пользователям:
* регистрироваться и входить в систему;
* создавать новые карточки с заголовком, подзаголовком и текстом;
* просматривать список всех своих карточек на главной странице;
* переходить к полной версии карточки для детального ознакомления;
* выходить из учётной записи.


Проект создан для демонстрации базовых принципов веб‑разработки: аутентификации, CRUD‑операций и шаблонизации.

## Технологии


* **Python** — язык программирования;
* **Flask** — веб‑фреймворк;
* **Jinja2** — шаблонизатор;
* **HTML5** — разметка интерфейса;
* **CSS3** — стилизация интерфейса;
* **SQLite** — база данных (для простоты развёртывания).

## Установка и запуск


### Предварительные требования

* Python 3.8 или выше;
* менеджер пакетов `pip`;
* любой современный браузер.

### Пошаговая инструкция

**Запустите приложение:**

* bash
  python main.py
